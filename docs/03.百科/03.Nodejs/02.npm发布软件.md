# npm发布软件
## 第一步、进入要发布的项目根目录，初始化为npm包：
```bash
npm init
```

## 第二步、注册npm用户
去官网注册[https://www.npmjs.com/signup](https://www.npmjs.com/signup)

需要邮箱验证，不然会报错


## 第三步、配置注册源
```bash
npm config set registry http://registry.npmjs.org 
```

## 第四步、登录
```bash
npm login
```

## 第五步、发布
```bash
npm publish
```
需要及时更新版本不然版本相同会发布失败。
