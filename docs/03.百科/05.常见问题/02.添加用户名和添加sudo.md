## 2.添加用户名和添加sudo
### 添加用户名
```bash
sudo useradd -m -d /data/username -s /bin/zsh username
```
### 修改密码
```bash
sudo passwd username
```
### 添加sudoers
```bash
vi /etc/sudoers
#添加用户
username    ALL=(ALL:ALL) ALL
```
