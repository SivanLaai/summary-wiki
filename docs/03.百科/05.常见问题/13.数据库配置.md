## mysql数据库
### mysql忘记密码
#### （1）修改配置文件免密
```
sudo vim /etc/mysql/my.cnf
```
```
[mysqld]
skip-grant-tables
```
#### （2）重启mysql
```
sudo service mysql restart
```
#### （3）修改初始密码
```
mysql -u root
use mysql;
update user set password=password("test@123") where user="root";
```
